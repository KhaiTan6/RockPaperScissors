# RockPaperScissors

This is a simple implementation of the classic Rock Paper Scissors game. It is built using HTML, and Javascript, and it allows you to play against the computer by clicking on buttons.

## How to play

1. Click on this link https://khaitan6.github.io/RockPaperScissors/.
2. Click on one of the buttons labeled "Rock", "Paper", or "Scissors" to make your choice.
3. The computer will randomly select its choice.
4. The result of the rould will be displayed, indicating wheter you won, lost or draw.
5. The scores will be updated after each round.
6. The first player to reach 5 points wins the game.
7. Click the "Reset" button to restart the game at any time.
